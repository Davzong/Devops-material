## Commnads between the local files and global files

将更改推送到远程仓库: 


推送到 master 分支
git push origin master      

9. 从远程仓库拉取更改:

git pull origin master      # 从 master 分支拉取

10. 创建分支:

git branch [branch-name]

11. 切换到其他分支:

git checkout [branch-name]

12. 合并分支:

git merge [branch-name]

13. 删除分支:

git branch -d [branch-name]

14. 查看所有分支:

git branch
